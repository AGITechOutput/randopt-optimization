RandOpt: Runtime Optimization for Large Model Inference
不拼架构、不堆参数，只做一件事：让已上线的模型变得更强。
纯推理层优化，在昇腾与英伟达平台实现精度与速度双提升。
核心进展
1. 动态分层噪声：精度再提 0.6 个百分点
- 策略：将噪声强度与网络深度绑定，浅层（前1/3）σ=0.005，中层 σ=0.01，深层 σ=0.02（区别于 MIT 原论文的固定噪声）
- 效果：
- 昇腾平台：精度从固定 σ=0.02 的 69.8% 提升至 70.4%，+0.6%
- 英伟达平台：同步验证，获得类似精度收益
- 优势：更贴合模型表达规律——浅层特征通用，少扰动；深层特征专用，多探索
- 后续目标：优化分层边界与噪声强度，目标再提 1% 以上
2. 昇腾并行扰动：速度提升 15%
- 策略：利用昇腾多流并行能力，将 5 个扰动模型分配至不同计算流同时推理
- 效果：
- 端到端速度 +15%，精度无损失，显存基本稳定
- RandOpt 在昇腾上的速度优势从 +5.2% 扩大至 +20% 以上
3. 英伟达侧两条线收口
- 动态集成（自适应 K 值）：速度波动基本抹平，精度微涨 +0.7%
- TensorRT 编译优化：迭代中，即将实现速度回正
- 累计收益：英伟达侧总精度提升达 10.9%（10.2% + 0.7%），速度不降反升
下一步方向
短期三条核心线（分层噪声、动态集成、TensorRT）均已跑通。
下一步将主攻 MoE 专家感知量化 + RandOpt 联动，目标实现：
- 显存占用降低 30%
- 推理精度提升 10%
该方向为行业尚未公开成果的前沿领域，进展将持续同步。
